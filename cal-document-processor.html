API Documentation

Simple REST API for programmatic access. All endpoints return JSON.

POST /upload

Upload Document

Upload a document for processing. Returns document ID for tracking.

Request

curl -X POST -F "document=@business-plan.md" http://localhost:3000/upload

Response

{
  "docId": "1756077217445",
  "message": "Document uploaded"
}
POST /process/{docId}

Process Document

Analyze document and determine best template match.

Request

curl -X POST http://localhost:3000/process/1756077217445

Response

{
  "type": "saas-platform",
  "features": ["authentication", "dashboard", "billing"],
  "template": "full-stack-saas",
  "confidence": 0.92
}
POST /generate/{docId}

Generate MVP Code

Generate complete, working application code based on analysis.

Request

curl -X POST http://localhost:3000/generate/1756077217445

Response

{
  "message": "Code generated",
  "files": ["package.json", "server.js", "public/index.html", "src/components/"],
  "downloadUrl": "/download/1756077217445"
}
