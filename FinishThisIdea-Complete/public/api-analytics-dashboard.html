API Documentation

POST /api/v1/generate
📊 12.3k calls/day ⚡ 234ms avg ✅ 99.8% success

Generate an MVP from a document. Supports multiple formats including Markdown, PDF, and chat logs. 

// Request
{
  "document": {
    "type": "markdown",
    "content": "# My SaaS Idea\\n\\nA platform for..."
  },
  "options": {
    "template": "saas-mvp",
    "deployment": "docker"
  }
}

// Response
{
  "success": true,
  "mvp": {
    "id": "mvp_123",
    "name": "My SaaS Platform",
    "downloadUrl": "https://api.example.com/download/mvp_123.zip"
  }
}
