Nginx acts as a traffic director, routing requests based on subdomain to the correct service port internally.

# Nginx configuration example
server {
  server_name app.documentgenerator.app;
  location / {
    proxy_pass http://localhost:8080;
  }
}

Summary

The unified domain system transforms the Document Generator from a collection of scattered services into a cohesive platform. Users access everything through intuitive subdomains, making the system feel professional and production-ready.

Key Points:

  • No more remembering port numbers
  • Clean, professional URLs
  • Easy to add SSL certificates
  • Room-based isolation with vanity URLs
  • WebSocket support for real-time features
  • Ready for production deployment
