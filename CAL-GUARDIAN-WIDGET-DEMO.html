One-Line Integration

Add Guardian to your website with just one line of code:

<script src="http://localhost:7777/api/stream/embed.js"></script>

Then add a div where you want the widget to appear:

<div class="guardian-widget" data-type="status" data-brand="deathtodata.com"></div>

üéØ Integration Steps

Add the script tag

Include the Guardian embed script in your HTML head or before closing body tag.

Add widget container

Place a div with class "guardian-widget" where you want the widget to appear.

Configure options

Use data attributes to customize: data-type, data-brand, data-theme, etc.

That's it!

The widget automatically initializes and connects to your Guardian system.

‚öôÔ∏è Configuration Options

Data Attributes

  • data-type: Widget type (status, activity, proof, ticker, floating)
  • data-brand: Your brand (deathtodata.com, soulfra.ai, or all)
  • data-theme: Color theme (dark or light)
  • data-show-proof: Show proof tab in activity widget (true/false)
  • data-animate: Enable animations (true/false)
  • data-auto-connect: Auto-connect to WebSocket (true/false)

JavaScript API

// Load widget programmatically const widget = GuardianWidget.load('#my-widget', { type: 'activity', brand: 'deathtodata.com', theme: 'dark', showProof: true }); // Verify proof chain GuardianWidget.verifyChain(); // Open full dashboard GuardianWidget.openDashboard('deathtodata.com');
