üìö How to Use

1. Start Playing:

- Click "Launch Arena"
- Enter your username
- Join a room (or create one)
- Use WASD to move, click to shoot

2. Start Recording:

- Press the record button in-game
- Select your screen to capture
- Play normally - highlights are auto-detected
- Stop recording to save your clip

3. Generate Trailer:

- After recording, click "Share Clip"
- Choose a trailer template
- The system auto-generates an epic trailer
- Download and share with friends!

üíª Integration Example

Here's how to integrate these systems into your own project:

// Initialize the game with recording
const game = new HyperCamArena();
const recorder = new HyperCamIntegration();
const multiplayer = new MultiplayerGameClient();

// Connect to multiplayer
await multiplayer.connect({
  name: 'PlayerName',
  roomId: 'arena-1'
});

// Start recording with effects
await recorder.startRecording({
  watermark: true,
  effects: true,
  quality: 'high'
});

// Generate trailer from recording
const trailer = await trailerGen.generateTrailer(
  recordingData,
  'epic'
);
